## es-covid-19

Index the latest covid-19 (coronavirus) outbreak data into Elasticsearch.

#### Usage

Run the following commands:

```
yarn install
node index
```

Then, if you have Kibana running, you can import `kibana-export.ndjson` for a dashboard:
